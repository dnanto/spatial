# spatial

trying out some spatial data structures with a quick-and-dirty 3D engine...

## OctTree

![octtree](octtree.png "octtree")

Key Commands:
|key|action|
|-|-|
|w/s|up/down|
|a/d|left/right|
|q/e|in/out|
|i/k|yaw|
|j/l|pitch|
|u/o|roll|
|v|toggle octants|
|b|toggle octants with points|
|n|toggle points|
|m|re-initialize random walk|
